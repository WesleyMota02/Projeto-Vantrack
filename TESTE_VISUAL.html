📋 Passos para Usar

1

Cadastre os Usuários

Use a API de cadastro ou insira manualmente no PostgreSQL usando as credenciais acima.

2

Acesse a Página de Login

Abra o navegador e acesse a página inicial do VanTrack.

3

Selecione o Tipo de Usuário

Clique no dropdown e escolha entre "Motorista" ou "Aluno".

4

Insira Email e Senha

Digite o email e senha fornecidos acima.

5

Clique em "Entrar"

Se for um novo dispositivo, você verá a página de 2FA pedindo um código de 6 dígitos.

6

Acesse o Dashboard

Após login bem-sucedido, você será redirecionado para o dashboard apropriado (motorista ou aluno).

🔐 Autenticação de Dois Fatores (2FA)

Quando você fizer login de um novo dispositivo, o sistema detectará e pedirá um código de verificação:

1

Detecção de Novo Dispositivo

O sistema analisa User-Agent e IP para identificar dispositivos novos.

2

Geração de Código

Um código de 6 dígitos é gerado e válido por 5 minutos.

3

Envio de Código

O código é enviado via SMS (Twilio) ou Email (SMTP).

4

Verificação

Digite o código na página 2FA. Máximo 3 tentativas erradas.

5

Dispositivo Marcado

Após verificação bem-sucedida, o dispositivo é marcado como confiável. Próximos logins não precisam de 2FA.

⚠️ Para testar 2FA: Limpe o localStorage ou use um navegador diferente/incógnito para simular um novo dispositivo.
